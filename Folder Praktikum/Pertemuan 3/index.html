A. Struktur Dasar HTML

Saya telah mempelajari bahwa HTML adalah bahasa markup, bukan bahasa pemrograman. HTML digunakan untuk membuat dan menyusun halaman web dengan berbagai elemen seperti teks, gambar, video, dan lain-lain.

Struktur dasar dokumen HTML terdiri dari:

  • <!DOCTYPE html> untuk mendefinisikan tipe dokumen
  • <html> sebagai elemen pembungkus utama
  • <head> untuk metadata seperti judul, link CSS, dan script
  • <body> untuk konten yang akan ditampilkan di browser
