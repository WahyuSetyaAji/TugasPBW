Catatan Belajar HTML - Pertemuan 4

1. Pengenalan Form dalam HTML

Form adalah elemen penting dalam HTML untuk mengumpulkan data dari pengguna. Form memiliki atribut action dan method (GET atau POST).

2. Input Text & Textarea

Input Text

Input teks berguna untuk data pendek seperti nama dan email.

Textarea

Textarea digunakan untuk masukan teks yang lebih panjang, seperti komentar atau deskripsi.

3. Dropdown (Combobox)

Dropdown digunakan untuk memilih satu dari beberapa pilihan.

4. Radio Button & Checkbox

Radio Button

Radio button memungkinkan pengguna memilih satu pilihan dari beberapa opsi.

Checkbox

Checkbox memungkinkan pengguna memilih lebih dari satu opsi.

5. Button & File Upload

Button

Tombol digunakan untuk mengirim form atau menjalankan aksi tertentu.

File Upload

<input type="file"> digunakan untuk mengunggah file dari komputer pengguna.

6. Validasi Form Dasar




Validasi form dapat dilakukan dengan atribut seperti required, min, max, pattern, dan sebagainya.
